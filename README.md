# WGU Network Engineering Labs

## Overview

This repository contains a collection of hands-on networking and security troubleshooting labs completed as part of my Bachelor of Science in Cloud & Network Engineering at Western Governors University (WGU).

The labs simulate real-world help desk, network engineering, and security support scenarios involving DNS, routing, firewalls, packet analysis, endpoint connectivity, and network security auditing.
